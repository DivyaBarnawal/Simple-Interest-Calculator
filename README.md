# 💰 Simple Interest Calculator

A lightweight and beginner-friendly **Bash-based Simple Interest Calculator** that computes simple interest using user-provided values for the principal amount, annual rate of interest, and time period.

This project was created to demonstrate basic shell scripting, Git version control, and GitHub collaboration practices.

---

## 📌 Overview

The Simple Interest Calculator allows users to quickly calculate the simple interest on a given principal amount using the standard mathematical formula.

It is designed as an educational project for learning:
- Bash scripting
- Git and GitHub workflow
- Open-source project structure
- Repository documentation

---

## ✨ Features

- Interactive command-line interface
- User-friendly input prompts
- Calculates simple interest instantly
- Lightweight and easy to run
- Beginner-friendly source code

---

## 🧮 Formula Used

```
Simple Interest = (Principal × Rate × Time) / 100
```

Where:

- **Principal (P)** = Initial amount of money
- **Rate (R)** = Annual interest rate (%)
- **Time (T)** = Time period (in years)

## 🚀 Getting Started

### Prerequisites

- Bash Shell
- Git (optional)

### Clone the Repository

```bash
git clone https://github.com/your-username/simple-interest-calculator.git
```

Navigate into the project folder:

```bash
cd simple-interest-calculator
```

Give execute permission:

```bash
chmod +x simple-interest.sh
```

Run the calculator:

```bash
./simple-interest.sh
```
